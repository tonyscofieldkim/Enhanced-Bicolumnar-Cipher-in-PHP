  • subjectWhat is EBC?

    EBC stands for enhanced bicolumnar cipher.
    Bicolumnar cipher basically encrypts plaintext by transposition, using two keys and uses the same keys to decrypt ciphertext. This module is Enhanced in that, instead of using the alphabet (A-Z), ASCII characterset is used, this means that we can work with special characters and control characters.
    This module is implemented in PHP with all the love by @Tony_Kimari.
    You can find the code for this module for free under MIT licence.

  • vpn_keyAbout the second key.

    Since bicolumnar cipher operates on text using two keys,In this context only, the first key provided is used to generate a second key.

    Generating the second key

    The second key is the result of the first key encrypting its reversed version. This generates a new key that has different characters ordering. The benefit of this is that the user only needs one key and it protects itself from known-plaintext cryptanalysis.

  • whatshotEnhanced mode

    In basic mode, ebc_encrypt() and ebc_decrypt() do not modify characters.
    This means the ciphertext contains exactly the same characters as the plaintext, just transposed. The integrity of the message is achieved by using the enhanced mode. i.e setting the third parameter to true

    1.//enhanced mode
    2. $ciphertext = ebc_encrypt ( $plaintext , $key , true ) ;
